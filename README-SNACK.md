# WisEnergy - Expo Snack Version

A placeholder version of the WisEnergy app for portfolio showcase on Expo Snack.

## How to Use

### Option 1: Direct Import (Recommended)
1. Go to [expo.dev/snack](https://expo.dev/snack)
2. Create new project
3. Copy ALL content from `WisEnergySnack.js`
4. Paste into `App.js`
5. Press "Run" to preview

### Option 2: GitHub Import
1. Go to [expo.dev/snack](https://expo.dev/snack)
2. Click "Import from GitHub"
3. Enter: `darinjan13/WisEnergy`
4. Select the file: `WisEnergySnack.js`
5. Press "Run"
